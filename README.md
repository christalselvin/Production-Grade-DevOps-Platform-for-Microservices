# Production-Grade DevOps Platform for Microservices

This project demonstrates how to build a **production-grade cloud-native DevOps platform** using modern DevOps tools.

## Microservices

- User Service
- Product Service
- Order Service
- Payment Service

## DevOps Stack

- Docker
- Kubernetes
- Jenkins
- Terraform
- Ansible
- ArgoCD
- RabbitMQ
- Prometheus
- Grafana
- JFrog Artifactory

## Features

- CI/CD pipeline automation
- GitOps deployment with ArgoCD
- Infrastructure as Code
- Kubernetes orchestration
- Monitoring and observability
- Event-driven microservices
